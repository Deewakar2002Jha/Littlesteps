# 🏥 LittleSteps

> A healthcare web platform for children receiving autism and speech/verbal disorder treatment.

Built for a real hospital to help parents easily book appointments and explore treatment services — making access to child healthcare simpler and stress-free.

🌍 **Live Demo** → https://littlesteps-seven.vercel.app

---

## 🎯 Purpose

LittleSteps was built to solve a real problem — parents of children with autism and verbal disorders often struggle to navigate hospital services and schedule the right appointments.

This platform makes that process simple, clear, and accessible.

---

## ✨ Features

* 📅 Appointment booking for child treatment sessions
* 🏥 Hospital services overview (autism therapy, speech therapy, etc.)
* 👨‍⚕️ Specialist/doctor information
* 📱 Fully responsive — works on mobile and desktop
* 🤖 AI-powered assistance via Google Gemini API

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Deewakar2002Jha/Littlesteps.git

# Navigate to project
cd Littlesteps

# Install dependencies
npm install

# Start development server
npm run dev
```
